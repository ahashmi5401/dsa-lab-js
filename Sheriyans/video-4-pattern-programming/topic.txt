# 📂 PATTERN PROGRAMMING: SETUP & LOGIC GUIDE

### 🛠️ 1. ENVIRONMENT SETUP (Node.js)
To take user input in the **Terminal** (not the browser), you must follow these steps:

* **STEP 1:** Open your terminal in this folder.
* **STEP 2:** Run `npm install prompt-sync` to download the input library.
* **STEP 3:** Use this exact syntax at the top of your `.js` files:
    > `const prompt = require('prompt-sync')();`

---

### 🚀 2. THE "SECRET SAUCE" OF PATTERNS
Pattern programming is essentially **Mapping a 2D Grid**.

* **Outer Loop (`i`):** Controls the **Vertical** movement (Rows). Think of this as the "Y-axis."
* **Inner Loop (`j`):** Controls the **Horizontal** movement (Columns). Think of this as the "X-axis."



---

### 💡 3. PRINTING STRATEGIES
There are two ways to handle the "same-line" printing requirement:

1.  **Direct Write (Node.js Only):**
    Use `process.stdout.write("*")`. This prints **without** jumping to the next line. Use a final `console.log()` after the inner loop to "Enter" to the next row.
    
2.  **String Building (Universal):**
    Create an empty string `let row = ""`, add characters to it in the inner loop (`row += "*"`), then `console.log(row)` once the row is finished.

---

### 📐 4. LOGIC CHEAT SHEET
The magic happens in the **Inner Loop Condition**. Change this to change the shape:

| Shape Target | Inner Loop Logic (`j` loop) | Result |
| :--- | :--- | :--- |
| **Square** | `j <= totalRows` | Fixed width |
| **Right Triangle** | `j <= i` | Width grows with row number |
| **Inverted Triangle** | `j <= (totalRows - i + 1)` | Width shrinks as row increases |



---

### ⚠️ COMMON ERRORS TO WATCH FOR
* **`ReferenceError: sync is not defined`**: You forgot the quotes! Use `require('prompt-sync')`.
* **`Infinite Loops`**: Always ensure your increment (`i++` or `j++`) is present.
* **`String vs Number`**: `prompt()` returns a string. Use `+prompt()` or `Number()` to convert it for math.
